# 🍅 トマトタイマー

ポモドーロテクニックを使った作業効率化ツール。

25分集中 → 5分休憩の繰り返しで、タスク管理と統計ダッシュボード付き。環境音プリセットで集中タイム中にBGMも流せます。

## 機能

- ⏱️ ポモドーロタイマー (25分集中 / 5分休憩 / 15分長休憩)
- 📝 タスク管理 (登録・完了チェック・ポモドーロ数記録)
- 📊 統計ダッシュボード (日/週/累計の集中時間)
- 🎵 環境音プリセット (雨の日のカフェ、森の中、海辺 など)
- 🔔 通知 (Notification API)
- 🌙 ダーク/ライトモード切替

## 使い方

`index.html` をブラウザで開くだけで動きます。サーバー不要。

## 技術

HTML / CSS / JavaScript / Web Audio API / LocalStorage
